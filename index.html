Come scaricare il report da Intercard

  1. Apri il software Back Office Intercard
  2. Seleziona Reports
  3. Entra in Revenue Reports
  4. Apri iTeller Sales Report
  5. Seleziona il range di date in cui vuoi verificare gli incassi (puoi prenderlo anche più ampio del necessario, poi filtri nell'app)
  6. Alla destra delle date clicca il pulsantino con le due frecce verdi in cerchio e attendi che si carichino i dati
  7. Clicca il tastino verde con il simbolo di Excel alla sinistra delle date per scaricare il file
  8. Torna qui e carica il file con il pulsante "Carica il file Excel"

Il file deve contenere le colonne: Location, Account, Form of Payment, Sale Amount, Card Value, QTY, Device, Date and Time.
